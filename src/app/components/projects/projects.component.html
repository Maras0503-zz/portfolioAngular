The calendar is an application created in Angular and it is a project that I'm currently working on. It's a simple calendar for planning meetings, appointments and setting up reminders.

MyStorage is an application created in JavaScript with use of jQuery and MySQL database connected via PHP. In this application, you can create invoices, hold information about the warehouse, create data related charts and much more.

CleanThisMess is an application created in JavaScript with use of jQuer and MySQL database connected via PHP. This application is for collecting information about waste generated in company production process. It's arranged by departments and products.
